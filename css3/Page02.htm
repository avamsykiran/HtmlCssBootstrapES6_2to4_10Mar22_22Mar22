About Web

A website is a collection of webpage that talk About a common topic and shar the same style and script. These web pages are expected to be internavigable.

A website can be either static or dynamic in nature . A static website is designed to serve information and the content of such sites is almost the same and will not change across different requests. Technologies employed to write static sites are :

  1. HTML 5
  2. CSS 3
  3. Bootstrap 5

A dynamic website has content that is generated by a server side program. the content of those pages is written by anybody but based on some parameters and data feteched from a database the page content is generated at runtime. Hence called the dynamic web application. Technologies employed here are:

  1. Server Side Programming Tools like
    1. Java - Servlets,Springs,Struts
    2. .Net - aspx,ASP.NEt
    3. PHP
  2. Client Side Scripting like javascript
